# OzzCompressor

A simple in-browser image compressor that targets a specific file size.

## How to use

1. Open `index.html` in a browser.
2. Drop an image or click to select one.
3. Enter the target size in KB.
4. Click Compress and download the result.

## Notes

- JPEG and WebP respect quality adjustments.
- PNG ignores quality and is best for lossless output.
- Everything runs locally in the browser.
