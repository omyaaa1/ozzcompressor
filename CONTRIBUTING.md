# Contributing

Thanks for your interest in improving OzzCompressor.

## Quick start

1. Fork the repo.
2. Make your change in a branch.
3. Ensure `npm run test` passes.
4. Open a pull request.

## Style

- Keep functions small and focused.
- Prefer plain JavaScript and browser APIs.
- Use meaningful names and avoid magic numbers.

## Suggestions

- Small PRs are easier to review.
- Add screenshots for UI changes.
