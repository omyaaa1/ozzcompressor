# OzzCompressor

A fast, in-browser image compressor that targets a specific file size.

## How to use

1. Open `index.html` in a browser.
2. Drop an image, click to select one, or paste from clipboard.
3. Enter the target size in KB.
4. Click Compress and download the result.

## Notes

- JPEG and WebP respect quality adjustments and size targeting.
- PNG is lossless and may not hit very small targets.
- Everything runs locally in the browser.
